# TableEditor

KPI course work

Using Kivy framework for building cross-platform applications.

Simple app for managing tables:
<ul>
    <li>create</li>
    <li>update</li>
    <li>save to file</li>
    <li>read from file</li>
</ul>

Open app directory.

Run:
```python main.py``` to start app.

Run: 
```buildozer android debug deploy``` to get *.apk file.
